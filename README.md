## 🎯 **Что мы создаем?**

Браузерную игру в жанре "rogue-like" (Vampire Survivors), где вы управляете героем, сражающимся с ордами нежити. Чем дольше выживаете — тем сильнее становитесь!

**Ключевые механики:**
- ⚔️ Автоматическая атака по врагам
- 💎 Сбор опыта для повышения уровня
- 🔥 8+ видов оружия с уникальными улучшениями
- 🧠 Сложная система прокачки с выбором

---

## 🎮 **Архитектура проекта (быстрый старт)**

```
project/
├── index.html          # Главный файл (с экраном загрузки)
├── css/
│   ├── style.css       # Основные стили + мобильная адаптация
│   ├── ui.css          # Стили HUD, карточек выбора
│   └── effects.css     # Анимации, флеши, партиклы
├── js/
│   ├── settings/       # Настройки (CONFIG, WEAPONS_DATA)
│   ├── graphics/       # Визуал (спрайты, частицы, UI)
│   └── game/           # Логика (игрок, враги, оружие)
└── assets/             # Картинки (спрайты, иконки)
```
